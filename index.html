COVID-19 in US

Framework of the PanViz system in modern web style.

Now I collected US-county-level data related to COVID-19. These features including: 1. Number of confirmed cases; 2. Confirmed number per 10K population; 3. Number of people confirmed per licensed hospital bed; 4. Number of deaths; 5. Death number per 10K population; 6. Daily new confirmed number; 7. Time series data of confirmed & death cases; 8. Each county's population & number of licensed beds.
